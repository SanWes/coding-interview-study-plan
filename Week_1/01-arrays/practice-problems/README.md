# 🧠 Arrays - Sliding Window & Prefix Sum Practice

This folder contains a curated set of practice problems designed to build intuition and skill with the **Sliding Window** technique — a fundamental pattern in array and string problems. Some problems also introduce **Prefix Sum** logic to prepare for slightly more complex variations.

---

## 📚 What You'll Learn

- How to use fixed-size and variable-size sliding windows
- Managing window expansion and contraction
- Prefix sum logic for running totals and efficient subarray calculations
- Pattern recognition to approach real-world interview-style problems
